2. **New Web Service**: Click "New +" â†’ "Web Service"
3. **Connect Repository**: 
   - Connect your GitHub account
   - Select `Subham-KRLX/Healthcare_Bot` repository

   - Add Disk: Mount path `/opt/render/project/src/instance`, Size: 1GB
6. **Deploy**: Click "Create Web Service"
7. **Wait**: 5-10 minutes for first deployment
8. **Done!** ðŸŽ‰ Your app will be live at `https://healthcare-bot-xxxx.onrender.com`
